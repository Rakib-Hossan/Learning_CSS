CSS Animation

CSS allows animation of HTML elements without using JavaScript. Their are two step to use animation:

1. Give a animation name using animation property.

2. Use @keyframes to set animation

Basic example of animation.

We change this div color yellow to green and text color turn into white. It changes within 5s.

Code for this basic animations are:

            
                .basic{
                    background-color: yellow;
                    width:50%;
                    margin:0 auto;
                    box-sizing: border-box;
                    padding: 50px;
                    animation-name: basic_example;
                    animation-duration: 5s;
                }
                @keyframes basic_example{
                    from{
                        background-color: yellow;
                    }
                    to{
                        background-color: blue;
                        color:white;
                    }
                }

This is second animation example

code for this animation

            
                .second{
                    width: 50%;
                    background-color: blueviolet;
                    padding: 20px;
                    color: white; 
                    text-align: center;
                    position: relative;
                }
                .second img{
                    position: absolute;
                    width: 30px;
                    height: 40px;
                    animation-name: second_img;
                    animation-duration: 5s;
                    animation-iteration-count: infinite;
                }
                @keyframes second_img {
                    0%{
                        bottom: 0;
                    }
                    25%{
                        bottom: 25%;
                        transform: scale(2) rotate(90deg);
                        opacity: 0.5;
                    }
                    50%{
                        bottom: 50%;
                        transform: scale(4) rotate(180deg);
                        opacity: 0.75;
                    }
                    75%{
                        bottom: 75%;
                        transform: scale(8) rotate(360deg);
                        opacity: 1;
                    }
                    100%{
                       bottom: 0;
                    }
                }
