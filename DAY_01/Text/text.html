Css text alignment

The text-align property is used to set the horizontal alignment of a text.

A text can be left or right aligned, centered, or justified.

The following example shows center aligned, and left and right aligned text (left alignment is default if text direction is left-to-right, and right alignment is default if text direction is right-to-left). All text alignment example are shown up.

Text Decoration

To style text we used text-decoration property. To decorate line we use text-decoration-line. To add color in text decoration we use text-decoration-color. To add style in text decoration we use text-decoration-style. To set text decoration thickness we use text-decoration-thickness. We can also set all text decoration property together we use text-decoration property.

Text transform

The text-transform property is used to specify uppercase and lowercase letters in a text.

THIS TEXT IS WRITTEN BY CAPITAL letter BUT TEXT-TRANSFORM PROPERTY CONVERT THIS LINE INTO SMALL letter. SYNTAX FOR THIS TEXT-TRANSFORM:LOWERCASE

this text is written by small letter but text-transform property convert this line into capital letter. syntax for this text-transform:uppercase

this text is written by small letter but text-transform property convert every word first letter is capital letter of this line. syntax for this text-transform:capitalized

Text spacing

Text spacing means space between text. To add spacing we used : text-indent; letter-spacing; line-height; word-spacing and white spacing property.

Text-indent

Text-indent property used to specify the indentation of the first line of a text. This line is the example. Syntax: text-indent:value;

Letter spacing

Letter-spacing property used to specify the space between the characters in a text.This line is the example. Syntax: letter-spacing:value;
