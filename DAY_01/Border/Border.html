Css Border width

The border-width property specifies the width of borders

CSS Border Style

The border-style property specifies what kind of border to display

dotted - Defines a dotted border

dashed - Defines a dashed border

solid - Defines a solid border

double - Defines a double border


Css Border Color

The border-color property is used to set the color of the four borders.

we set this paragraph color in green

Specific Side Color

The border-color property can have from one to four values

we set right border color into red

Same way we can change all border side color. we also set four different color in one element

CSS border radius

The border-radius property is used to add rounded borders to an element
