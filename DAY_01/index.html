What is CSS?

CSS stands for Cascading style sheet. CSS describe for how html elements are to be displayed on screen.

Syntax of CSS

selector_name{property:value}

CSS Selector

CSS Selector are used to find HTML element which we want to style.

css element selector

The element selector select HTML element based on the element name. example if want change h3 text color we write: h3{color:green}

css id selector

The id selector uses the id attribute of a HTML element to select a specific element. To select an element with a specific id, write a hash character, followed by the id of the element. Example: suppose we set a h2 element id is heading_text and we want change background color of this particular id we write code: #heading_text{background:red}

css class selector

The class selector uses the class attribute fo a Html element to select a specific element. To select an element with a specific class, write a dot character, followed by the class of the element. Example: suppose we set a p element class is des and we change font-wight of this particular class we write code: .des{font-weight:700}

css universal selector

The universal selector selects all HTML element on the page. To select all html element in css we use asterisk. Example: suppose we want to set padding all html element we write: *{padding:20px}

Css grouping Selector

The grouping selector selects all the HTML elements with the same style definitions. Example: suppose we want to align text in h2,p tag. we write: h2,p{text-align:center}

How to Add css

Three ways to insert CSS. such as: 1.inline CSS 2.internal CSS 3. External CSS.

CSS Comments

Comments are used to explain the code. CSS comments are not displayed in the browser. Syntax: /* contents*/

CSS color

Colors are specified using predefined color names or RGB, HEX, HSL, RGBA, HSLA values.

By color name: color: red

By RGB code: rgb(255,0,0)

By HEX code: #ff0000

By HSL code: HSL(0,100%,50%)

By RGBA code: rgba(255,0,0,1)

By HSLA code: HSLA(0,100%,50%,1)

Css Margin

Margins are used to create space around elements, outside of any defined borders.

CSS has properties for specifying the margin for each side of an element:margin-top; margin-right; margin-bottom; margin-left


CSS Padding

Padding is used to create space around an element's content, inside of any defined borders.
