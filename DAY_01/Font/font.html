CSS google font

If we do not want to use any of the standard fonts in HTML, we can use Google Fonts. Google Fonts are free to use, and have more than 1000 fonts to choose from.To use google font we want add the link of google font on our html head tag then we use font-family property to set google font. Here we use Dancing Script font.

Font shorthand

To shorten the code, it is also possible to specify all the individual font properties in one property. Syntax: font:font-style font-weight font-size font-family;
